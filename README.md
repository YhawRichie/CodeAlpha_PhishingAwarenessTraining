# Phishing Awareness Training

## Overview
This project was developed as part of the CodeAlpha Cyber Security Internship.

The project is an interactive phishing awareness training module designed to educate users about phishing attacks, social engineering techniques, fake websites, and cybersecurity best practices.

## Topics Covered

* Introduction to Phishing
* Types of Phishing Attacks
* Recognizing Phishing Attempts
* Fake Websites and URL Tricks
* Social Engineering Tactics
* Prevention and Protection Methods
* Real-World Case Studies
* Interactive Quiz

## Features
* Interactive PowerPoint Module
* Knowledge Assessment Quiz
* Real-World Examples

## Tools Used
* Microsoft PowerPoint
* Cybersecurity Research
